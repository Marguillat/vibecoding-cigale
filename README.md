# Vibecoding CRM la Cigale
## Lancer le projet:
- Variables d'environnement dans un fichier `./web/env.local` :
```env
AIRTABLE_API_KEY=$$$$$$$$$$$$$$
AIRTABLE_BASE_ID=$$$$$$$$$$$$$$
AIRTABLE_TABLE_NAME=reservations
APP_ACCESS_CODE=1234
JWT_SECRET=
```
- Installer les dépendances depuis la racine
```bash
cd ./web && npm install
```

- Lancer l'exécution depuis la racine
```bash
cd ./web && npm run dev
```
