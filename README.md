# Vibecoding CRM la Cigale
## Lancer le projet:
- Variables d'environnement dans un fichier `./web/env.local` :
```env
AIRTABLE_API_KEY=$$$$$$$$$$$$$$
AIRTABLE_BASE_ID=$$$$$$$$$$$$$$
AIRTABLE_TABLE_NAME=$$$$$$$$$$
APP_ACCESS_CODE=$$$$
JWT_SECRET=
```
- Lancer l'exécution depuis la racine
```bash
cd ./web && npm run dev
```
